# Movie App Backend

## Overview
This is the backend for the Movie App, which includes user authentication features like **Login**, **Registration**, **Forgot Password**, and **Reset Password**. The backend uses **Node.js**, **Express**, and **MongoDB** to handle requests and store data.

## Features
- **User Registration**: Allows new users to register with email and password.
- **Login**: Authenticates users with their email and password.
- **Forgot Password**: Sends a password reset link to the userâ€™s email.
- **Reset Password**: Allows users to set a new password after clicking the reset link.

## Technologies
- **Node.js**: JavaScript runtime to build the backend server.
- **Express**: Web framework for Node.js.
- **MongoDB**: Database to store user data.
- **Nodemailer**: Service to send email notifications (e.g., for password resets).
- **JWT**: JSON Web Tokens for secure user authentication.
